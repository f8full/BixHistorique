bixhistorique-2015

Parsed and recorded 'https://montreal.bixi.com/data/bikeStations.xml' every 5 minutes between April 15th, 00:01 and November 15th, 23:59 September 29th, 08:24*, Montréal time.
Data before May 11th, 2015 13h13 is unreliable**

For more stats about the stored data, including licensing, see here.

To see the currently available API click here.


For finer, down to the minute data sets served through the same API, see URLs for each month

  • April - no dataset (stalled data)** ~26000 parse
  • May - partial dataset (stalled data)**
  • June - complete dataset 42229 parse
  • July - complete dataset 43678 parse
  • August - complete dataset 43752 parse
  • September - truncated dataset 40056 parse*
  • October - empty*
  • November - empty*

The source code of this web application is available on github.

*bike system backend was updated on September 29th, terminating any further availability recording.

**bike system backend was misconfigured. Data served over http presented incorrect cache-control headers. Workaround deployed on May 11th, 2015 13h13 EST

As of February 25th, 2016 serving over http is fixed with correct no-cache, no-store, must-revalidate, max-age=0, proxy-revalidate, no-transform headers.
