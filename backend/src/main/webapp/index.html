bixhistorique-2015

Parsing 'https://montreal.bixi.com/data/bikeStations.xml' every 5 minutes when parsing_active is true. to set parsing_active to true, use the URL '/cron/activateparsingcronjob?activate=true' as an admin. To set parsing_active to false, simply make a call to '/cron/activateparsingcronjob?activate=false'
